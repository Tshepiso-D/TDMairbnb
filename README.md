# TDMairbnb
This is a clone of airbnb
A full-stack Airbnb-inspired accommodation booking platform built with React, Vite, Node.js, Express, and MongoDB. Users can browse listings, view property details, authenticate, and manage bookings through a modern responsive interface.

Live Demo: ZAIO Airbnb Clone

Features
User authentication (Login / Registration)
Browse available properties
Property detail pages
Responsive design for desktop and mobile
REST API integration
MongoDB database persistence
Protected routes for authenticated users
Modern React frontend built with Vite
Tech Stack
Frontend
React
React Router
Axios
Vite
Backend
Node.js
Express.js
MongoDB
Mongoose
Deployment
Frontend: Netlify
Backend: Heroku (or your chosen backend host)
